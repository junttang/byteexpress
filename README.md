# ByteExpress

This repository contains ByteExpress, the proposed technology introduced in the paper  
**"ByteExpress: A High-Performance and Traffic-Efficient Inline Transfer of Small Payloads over NVMe"**,  
submitted to **HotStorage 2025**.

As of **April 6**, we are currently organizing the codebase and developing automation scripts in preparation for open-sourcing.

Thank you for your interest!
